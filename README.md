## Run for development

Install dependencies:

```
npm i
```

### Run unit tests

```
npm test
```

Watch mode:


```
npm run test:watch
```

Coverage report with:

```
npm run test:coverage
```

### Run linter

```
npm run lint
```

Auto-fix:

```
npm run lint:fix
```

### Build for production

```
npm run build
```
